ECG Simulator

During my first coop I was in a group with five other biomedical engineering students and we were tasked with creating an ECG Simulator within three months. This ECG Simulator would be used to test out the Holter devices at the company to see if they could pick up heartbeat signals properly.

1. Research
We researched different strategies and methods to build this device. I researched the different ISO standards that would be implemented for this device based on its purpose. The ISO standards we used for our reference were IEC 60601-1 and ISO 14971. These correlate to Basic Safety and Essential Performance for Medical Devices and Risk Management for Medical Devices. We drafted a digikey list that we gave to management for the components needed to build this simulator. To build the simulator we used a

  • Breadboard
  • Adafruit Feather Arduino
  • Digital-to-Analog converter
  • Resistors
  • Insulated Copper Wire
We bought other items that we weren't able to implement into the circuit due to timing and lack of resources. These are
  • 5k Potentiometer Panel Control
  • Knob ¼” shaft
  • COM-09765 7 segment display

     

2. Constructing the circuit

We went to the University of Waterloo to solder the pins on the arduino and construct the circuit board. We used our research as reference on how to construct the circuit while incorporating the different pieces that we had.

3. Designing the case
We wanted to design a case to house the circuit board so that anyone using it wouldn't be exposed to wiring and could use the components they needed without any confusion. The case was designed so that it could be opened and the circuit board could be taken out if it needed any maintenance. It would had grooves and ridges that correlated to the breadboard so that the bread board would fit in these indentations/extrustions to prevent it from moving in the case as the case is being transported. The pieces of the case that were supposed to be designed split between a group member and I and we both created them on SolidWorks. I created the top half and they created the bottom with both of us helping the other when creating the case. The case was assembled on SolidWorks Assembly to ensure that the dimensions of both pieces matched.

4. Code
We adapted code from the research we used to start this project. We had to alter the code because there were variations for the device this code was used for veruses what we were building. Due to a lack of time, we were not able ot prefect the code for the device but the code it work and it did show some readings on the holter device. This meant it was emitting millivolts to the Holter monitor.
