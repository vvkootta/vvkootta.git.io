Personal Project

This personal project was done when I was in high school in the manufacturing class room of my school. Using CorelDraw, I learned to import images into the program and convert them into vectors. Then I manipulated the vectors based on how small a support piece of metal could be without breaking apart when it was cut with the CNC Machine. I had to problem solve and transform each design so that they could be recreated using metal and empty space. This meant that I had to visualize the minimal amount of metal needed to create a design. I experimented with the free space in my designs to create shapes when creating these nameplates.

After creating the designs, I uploaded them into the CNC machine software and programmed it to cut out the different designs I had. After, I used the hand grinder to smooth the surface of the nameplate. This eliminated any imperfections that occurred when the metal was cut and gave it a shiny surface.

The original plan was to create a hanging mobile with these different pieces but I decided to keep the pieces separate so that my cousins would be able to keep their piece. Finally, all the nameplates were sent to be powder coated with different colours.
